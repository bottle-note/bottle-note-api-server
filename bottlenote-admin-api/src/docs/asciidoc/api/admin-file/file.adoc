=== PreSigned URL 발급 ===

- S3 업로드용 PreSigned URL을 발급합니다.
- 발급된 URL은 지정된 시간(기본 5분) 동안 유효합니다.
- 여러 개의 URL을 한 번에 발급할 수 있습니다.

[source]
----
GET /admin/api/v1/s3/presign-url
----
