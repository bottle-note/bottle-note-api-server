[discrete]
===== _정렬 방향 (sortOrder)_ =====

- 해당 정의는 정렬 방향을 나타내는 값입니다.
- 서비스의 모든 정렬 관련 API에서 공통적으로 사용됩니다.

[cols="1,1,3",options="header"]
|===
|  | |
| DESC
| 내림차순
| 5 -> 3 -> 2 -> 1

| ASC
| 오름차순
| 1 -> 2 -> 3 -> 5
|===

- 랜덤의 경우 정렬 타입에 따라 의도와 다르게 동작 할 수 있습니다.
- 해당 값이 없을 경우 기본값은 DESC(내림차순) 입니다.
- 타입이 잘못 전달될 경우 예외가 반환됩니다.
