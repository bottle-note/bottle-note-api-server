=== 팔로잉 리스트 조회 ===

팔로잉 유저 목록 리스트 조회 기능입니다.

요청 경로에 조회할 유저의 식별자가 포함되어야 합니다.
유저의 식별자는 Long 타입입니다.

|====
|PathVariable |설명

|userId | 유저 ID

|====

API EndPoint

팔로잉 리스트 조회 : 조회 대상 사용자가 팔로잉한 유저목록 정보를 조회합니다.

[source]
----
GET /api/v1/follow/{userId}/following-list
----
