=== Apple 로그인용 Nonce 발급 ===

- Apple 로그인 프로세스에서 사용할 일회성 Nonce 값을 발급합니다.
- 이 Nonce는 Apple 로그인 시 보안을 위해 사용됩니다.

[source]
----
GET /api/v2/auth/apple/nonce
----

=== Apple 로그인 ===

- Apple ID를 통한 로그인을 수행합니다.
- Apple에서 발급받은 ID Token과 이전에 발급받은 Nonce 값이 필요합니다.
- 성공 시 액세스 토큰을 반환하고, 리프레시 토큰은 HttpOnly 쿠키로 설정됩니다.

[source]
----
POST /api/v2/auth/apple
----
