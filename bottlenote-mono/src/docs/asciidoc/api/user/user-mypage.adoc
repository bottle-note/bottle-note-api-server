=== 마이페이지 기본정보 조회 ===

마이페이지 접근시 기본정보를 조회합니다.

조회할 수 있는 정보는

* 대상 사용자의 프로필 이미지
* 대상 사용자의 닉네임
* 대상 사용자를 팔로우한 유저 수 (팔로워 수)
* 대상 사용자가 팔로우한 유저 수 (팔로잉 수)
* 대상 사용자가 위스키를 찜하기 한 갯수
* 대상 사용자가 위스키에 작성한 리뷰 갯수
* 대상 사용자가 위스키에 평가한 별점의 갯수
* 로그인 사용자가 대상사용자를 팔로워 하는지 유/무
* 로그인 사용자와 마이페이지 대상 사용자가 같은 사용자인지 유/무

[source]
----
GET /api/mypage/{userId}
----
