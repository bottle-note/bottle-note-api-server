=== 유저 신고 요청 ===

특정한 사용자가 부적절한 행동을 했을 때, 해당 사용자를 신고할 수 있습니다.

user report의 경우 사용자간 신고로,

신고자와 피신고자가 모두 사용자인 경우에만 사용할 수 있습니다.

[cols="1,2",options="header"]
|===
| 신고 유형 코드 | 설명

| `SPAM`
| 스팸

| `INAPPROPRIATE_CONTENT`
| 부적절한 콘텐츠

| `FRAUD`
| 사기

| `COPYRIGHT_INFRINGEMENT`
| 저작권 침해

| `OTHER`
| 기타
|===

_추가되면 좋을 것 같은 신고 유형이 있으면 요청 바랍니다._

'''

[source]
----
POST /api/v1/user-report
----

|===
| 코드 |  예외 상황 | 에러 메시지

| 400
| `content`가 300자를 초과하는 경우
| "`content`는 300자 이내여야 합니다."

| 400
| 신고자가 자신을 신고하는 경우
| "자신을 신고할 수 없습니다."

| 400
| 동일 유저에 대해 일당 최대 한건의 신고를 초과하는 경우
| "동일 유저에 대해 일당 최대 한건의 신고만 가능합니다."

| 400
| 신고 대상 사용자가 일일 신고 횟수 제한을 초과하는 경우
| "일일 신고 횟수 제한을 초과하였습니다."

| 404
| 신고 대상 사용자가 존재하지 않는 경우
| "신고 대상 사용자가 존재하지 않습니다."

|===

** **추가되지 않은 에러 코드는 요청 바랍니다.**
** **request valid에 대한 내용은 공통 예외를 통해 처리됩니다.**

==== Error code ====

* 해당 API 요청 시 발생할 수 있는 예외를 설명 합니다.

[cols="1,1,1",options="header"]
|===
| 코드 | 상태 | 설명
| REQUIRED_USER_ID
| BAD_REQUEST
| 유저 아이디는 필수입니다.

| REPORT_TARGET_USER_ID_REQUIRED
| BAD_REQUEST
| 신고 대상자 아이디는 필수입니다.

| REPORT_TYPE_NOT_VALID
| BAD_REQUEST
| 신고 타입이 적절하지 않습니다. ( SPAM , INAPPROPRIATE_CONTENT " +
",FRAUD ,COPYRIGHT_INFRINGEMENT ,OTHER )

| CONTENT_NOT_BLANK
| BAD_REQUEST
| 공백입니다. 내용을 입력해주세요.

| REPORT_CONTENT_MAX_SIZE
| BAD_REQUEST
| 신고 내용은 300자 이내로 작성해주세요.

| IS_NO_USER_ID_TO_FOLLOW
| BAD_REQUEST
| 팔로우 할 유저의 아이디가 없습니다.

| SELF_REPORT
| BAD_REQUEST
| 자신을 신고할 수 없습니다.

| USER_NOT_FOUND
| BAD_REQUEST
| 유저를 찾을 수 없습니다.

| REPORT_USER_NOT_FOUND
| BAD_REQUEST
| 신고 대상 유저를 찾을 수 없습니다.

|===
