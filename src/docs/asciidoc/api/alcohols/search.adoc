=== 술(위스키) 목록 조회 ===

검색 조건으로 술(위스키) 정보를 조회합니다.

검색어를 통해 필터링 가능한 내역

* 위스키 이름 (한글/영문)
* 카테고리 (한글/영문)
* 지역 (한글/영문)
* 테이스팅 태그 (한글/영문)
* 시즌별 추천 키워드
- `봄 추천 위스키` : 봄철에 어울리는 위스키 추천
- `여름 추천 위스키` : 여름철에 어울리는 위스키 추천
- `가을 추천 위스키` : 가을철에 어울리는 위스키 추천
- `겨울 추천 위스키` : 겨울철에 어울리는 위스키 추천
- `비 오는 날 추천 위스키` : 비오는 날에 어울리는 위스키 추천
