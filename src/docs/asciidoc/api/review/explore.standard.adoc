=== 리뷰 둘러보기(standard) API  ===

- 기본형 리뷰 둘러보기 API로, 사용자들이 작성한 리뷰를 탐색할 수 있습니다.
- 리뷰는 최신순(수정 시간 기준)으로 정렬되어 제공됩니다.
- 검색 키워드는 작성자 이름, 술 이름, 리뷰 내용, 테이스팅 태그를 대상으로 합니다.
- 여러 키워드 입력 시 AND 조건으로 검색됩니다(모든 키워드를 포함하는 리뷰만 결과에 표시).

[source]
keywords="keyword1"&keywords="keyword2"
- 형태로 여러 키워드를 입력할 수 있다.
