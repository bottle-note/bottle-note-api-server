=== 팔로우/팔로워 리스트 조회 ===

팔로우 리스트 조회 기능입니다.

요청 경로에 조회할 유저의 식별자가 포함되어야 합니다.
유저의 식별자는 Long 타입입니다.

|====
|PathVariable |설명

|userId | 유저 ID

|====

API EndPoint

팔로잉/팔로워 리스트 조회 : 조회 대상 사용자가 팔로우한 유저들과 조회 대상 사용자를 팔로우하고 있는 유저들의 정보를 조회합니다.

[source]
----
GET /api/v1/follow/{userId}/relation-list
----
