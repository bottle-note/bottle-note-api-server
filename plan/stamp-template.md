# PROJECT COMPLETION STAMP Template

플랜 문서 마무리 시 문서 상단에 추가할 스탬프 템플릿입니다.

---

## 완료된 프로젝트

```
================================================================================
                          PROJECT COMPLETION STAMP
================================================================================
Status: **COMPLETED**
Completion Date: YYYY-MM-DD

** Core Achievements **
- [주요 성과 1]
- [주요 성과 2]
- [주요 성과 3]

** Key Components **
- [핵심 컴포넌트 1: 클래스명, 경로]
- [핵심 컴포넌트 2: 역할]

** Deferred Items ** (선택사항)
- [보류 항목: 무엇을, 왜]
================================================================================
```

---

## 진행 중인 프로젝트

```
================================================================================
                          PROJECT COMPLETION STAMP
================================================================================
Status: **IN PROGRESS**
Start Date: YYYY-MM-DD
Last Updated: YYYY-MM-DD

** Completed Work **
- [완료된 작업 1]
- [완료된 작업 2]

** Lessons Learned **
- [학습 내용 1]
- [학습 내용 2]

** Remaining Work **
- Phase N: [남은 작업 1]
- Phase N: [남은 작업 2]

** Key Principle ** (선택사항)
- [핵심 원칙 1]
- [핵심 원칙 2]
================================================================================
```

---

## 작성 원칙

- 구체적 수치 포함 (N개, X%, 파일명)
- 각 항목 1-2줄 요약
- 파일 경로는 전체 경로 명시
- 한글 프로젝트는 한글로 작성
