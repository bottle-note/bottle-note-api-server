=== 홈 배너 목록 조회 ===

홈 화면에 표시할 활성화된 배너 목록을 조회합니다.

배너는 `sortOrder` 기준 오름차순으로 정렬되어 반환됩니다.

[source]
----
GET /api/v1/banners?limit={limit}
----

[discrete]
==== TextPosition (텍스트 위치) ====

배너 내 텍스트가 표시될 위치를 지정합니다.

[cols="1,2",options="header"]
|===
| 코드 | 설명

| LT
| 좌측 상단 (Left Top)

| LB
| 좌측 하단 (Left Bottom)

| RT
| 우측 상단 (Right Top)

| RB
| 우측 하단 (Right Bottom)

| CENTER
| 중앙 (Center)
|===

[discrete]
==== BannerType (배너 유형) ====

배너의 목적에 따른 유형을 구분합니다.

[cols="1,2",options="header"]
|===
| 코드 | 설명

| SURVEY
| 설문조사 배너

| CURATION
| 큐레이션 배너 (추천 콘텐츠)

| AD
| 광고 배너

| PARTNERSHIP
| 제휴 배너
|===

[discrete]
==== 참고 사항 ====

* 비활성화된 배너(`isActive=false`)는 조회되지 않습니다.
* `startDate`/`endDate`가 null인 경우 상시 노출 배너입니다.
* `isExternalUrl=true`인 경우 새 창(또는 외부 브라우저)으로 열어야 합니다.
* 기본 `limit`은 10이며, 홈 화면에서는 최대 10개까지 노출을 권장합니다.
